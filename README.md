# BestOffer

# O que é?
Um projeto criado para poder ajudar a comparar e ver em qual lugar seu tão desejado produto está mais barato ou mais bem avaliado.

# O que estarei usando?
- HTMLg
- CSS (Visual)
- JS (Script de interação)
- Python (Webscrapping)

# O que esse projeto ajuda?
- Webscrapping: pegar conteúdo de diversas páginas webs e mostrá-los de forma intuitiva e fácil para usuários
- JS: cards gerados de forma dinâmica e automática dependendo dos resultados de busca
